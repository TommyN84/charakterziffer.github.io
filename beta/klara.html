Aus den Archiven: Klara

Die folgende Serie entstand 2011 im Rahmen eines Wettbewerbs für Kurzhörspiele. Die Stücke durften laut Vorgabe nur 140 Zeichen lang sein – perfekt, um sie auch zu twittern. Um den beschränkten Text trotzdem lang erscheinen zu lassen, habe ich mit Wiederholungen gearbeitet, wodurch die Tweets auch einen analytischen und mantra-artigen Charakter bekommen.

Thema der Serie sind Liebesschwüre und deren abgegriffene Komplimente. Wenn man die verbrauchten Vergleiche steigert und detaillierter formuliert, kippen sie vom Banalen ins Absurde, Lustige oder Poetische.

◎ Klara hat Augen // Klara hat blaue Augen // Klara hat himmelblaue Augen // Sturmhimmelblau. (#)

ツ Klara hat ein Lächeln // Klara hat ein süßes Lächeln // Klara hat ein zuckersüßes Lächeln // Hagelzuckersüß. (#)

ↁ Klara hat einen Bauch // Klara hat einen runden Bauch // Klara hat einen kugelrunden Bauch // Mozartkugelrund. (#)

♪ Klara hat eine Stimme // Klara hat eine helle Stimme // Klara hat eine glockenhelle Stimme // Osterglockenhell. (#)

❀ Klara hat eine Haut // Klara hat eine zarte Haut // Klara hat eine blütenzarte Haut // Apfelblütenzart. (#)

♡ Klara hat ein Herz // Klara hat ein warmes Herz // Klara hat ein ofenwarmes Herz // Kaminofenwarm. (#)

◛ Klara hat Lippen // Klara hat rote Lippen // Klara hat feuerrote Lippen // Lagerfeuerrot. (#)

✫ Klara hat Finger // Klara hat flinke Finger // Klara hat wieselflinke Finger // Mauswieselflink. (#)

≋ Klara hat Haare // Klara hat blonde Haare // Klara hat honigblonde Haare // Akazienhonigblond. (#)
